Installation

安裝六步，三分鐘上線

PoC v0 沒走 Apple Notarization，第一次需要繞 Gatekeeper（步驟 4）。v1 上線後會通過正式公證，雙擊即用。

  1. 01

    下載 DMG

    點上方按鈕下載 NClawPro-PoC-v0.dmg（488 KB）。

  2. 02

    雙擊掛載

    雙擊 dmg 檔開啟，會看到 N+Claw Pro 圖示。

  3. 03

    拖到 Applications

    把 N+Claw Pro 拖到右邊的 Applications 資料夾。

  4. 04

    繞過 Gatekeeper（首次）

    雙擊 N+Claw Pro 第一次會看到「無法打開」對話框。請進「系統設定 → 隱私與安全性」拉到最下方，點「仍要打開」並輸入密碼。如還不行，開 Terminal 貼：
    xattr -cr "/Applications/N+Claw Pro.app"

  5. 05

    首次啟動

    自動安裝 OpenClaw 與配置（需要 Node.js，沒裝會引導你下載）。

  6. 06

    開始用

    瀏覽器自動開啟 N+Claw Pro Control UI，可以開始 chat。
